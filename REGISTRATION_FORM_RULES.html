user registration portal
  • Users without login credentials for internet access can be allowed to request for access by filling up a form.
  • If auto-approval of requests is disabled, the administrators are notified by email and a notification appears on the dashboard when a user requests for access. The administrator can review the request and choose to approve or disapprove the request.
  • The details which the user has to provide and how the request has to be handled when some one requests for internet access can be configured on this page.
  • All changes will apply only for users who register hereafter and not to users who already have an account.
